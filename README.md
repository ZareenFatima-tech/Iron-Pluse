# Iron Pulse - Gym Management System

A comprehensive, full-stack Gym Management Web Application featuring separate panels for Users and Admins, real-time data synchronization, and online payment integration.

*   **Admin Panel**: Full CRUD for Gyms (Add/Edit/Delete), Manage Bookings (Approve/Reject), and track Revenue/Analytics.
*   **Real-Time**: Built with `Socket.IO` for instant updates across all connected clients without page reloads.
*   **Payments**: Integrated simulation for JazzCash and SadaPay.
